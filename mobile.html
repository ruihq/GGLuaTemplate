Starting hook

The starting hook is used to initialize the script and display a loading message.


gg.toast("Starting hook...");
gg.sleep(2250);

Welcome Phase

The welcome phase displays the initial alerts and loading progress.


gg.setVisible(false);
gg.alert("RHQ Script Template", "「START」");
gg.clearResults();

gg.sleep(500);

Menu

The menu displays the available options and triggers the corresponding functions based on the user's choice.


menu = gg.choice({
    "Option 1",
    "Option 2",
    "Option 3",
    "Option 4"
}, nil, 'Home Menu - RHQ Script Template');

if menu == 1 then option1() end;
if menu == 2 then option2() end;
if menu == 3 then option3() end;
if menu == 4 then option4() end;

Functions

The functions contain the code for each option in the menu.


function option1() {
    gg.alert("Option 1", "OK");
}

function option2() {
    gg.alert("Option 2", "OK");
}

function option3() {
    gg.alert("Option 3", "OK");
}

function option4() {
    gg.alert("Option 4", "OK");
}
